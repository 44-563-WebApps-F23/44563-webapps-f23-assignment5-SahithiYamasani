When you decrease the page's width, the second row containing the city names is more likely to break or wrap to the next line first. This is because these city name elements are placed within larger col-md containers. In contrast, the first row, which displays letters, should remain relatively stable on smaller screens. This stability is achieved by using col-sm-2 containers for the letter elements, which allows them to maintain their layout without breaking as easily.
